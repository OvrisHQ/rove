⚠️ Authentication Required

All API endpoints require Bearer token authentication. First, obtain a token from the /api/auth endpoint.

Authentication

POST /api/auth

Obtain an authentication token for API access.

curl -X POST http://localhost:8080/api/auth \
  -H "Content-Type: application/json" \
  -d '{}'

Response:

{"token": "abc123..."}

Task Management

POST /api/tasks

Submit a new task for execution.

curl -X POST http://localhost:8080/api/tasks \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"task": "List files in current directory"}'

Response:

{"success": true, "task_id": "task_123"}
GET /api/tasks/:id

Get the status of a specific task.

curl http://localhost:8080/api/tasks/task_123 \
  -H "Authorization: Bearer YOUR_TOKEN"

Response:

{"id": "task_123", "input": "...", "status": "completed", "created_at": 1234567890}
GET /api/tasks

Get task history (last 10 tasks).

curl http://localhost:8080/api/tasks \
  -H "Authorization: Bearer YOUR_TOKEN"

Response:

{"success": true, "tasks": [...]}
DELETE /api/tasks/:id

Cancel a running task (not yet implemented).

curl -X DELETE http://localhost:8080/api/tasks/task_123 \
  -H "Authorization: Bearer YOUR_TOKEN"

Server Status

GET /api/status

Get server status and version information.

curl http://localhost:8080/api/status

Response:

{"status": "running", "version": "0.1.0", "server_type": "api"}
